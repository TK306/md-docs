# 要件定義書

## 要件リスト

* REQ-01: md-docsは、Markdownファイルを任意の構造体に相互変換することのできるツールを作成するための機能を提供する。
* REQ-02: md-docsのユーザコードで定義する構造体は、md-docsで定義される抽象クラスを継承したdataclassとして利用できるようにする。
* REQ-03: ユーザーコードで定義される構造体とマークダウンは直接変換するのではなく、md-docsで定義するマークダウン部品モデルを介して実施する。
* REQ-04: ユーザーコードでは、ユーザーの構造体とマークダウン部品モデルの相互変換を実装するが、この実装が最小限となるように設計する。
